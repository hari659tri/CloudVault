<h1 align="center">☁️ CloudVault</h1>

<p align="center">
  A Full-Stack Cloud File Management System inspired by Google Drive 🚀
</p>

<hr/>

<h2>📌 Overview</h2>
<p>
CloudVault is a full-stack cloud-based file management application that allows users to securely upload, manage, and delete files. 
It is inspired by Google Drive and built using modern web technologies with a focus on security, scalability, and performance.
</p>

<hr/>

<h2>✨ Features</h2>
<ul>
  <li>🔐 User Authentication (JWT & bcrypt)</li>
  <li>📁 Secure File Upload & Deletion</li>
  <li>☁️ Cloud Storage Integration (Cloudinary)</li>
  <li>📦 RESTful API Architecture</li>
  <li>📄 Dynamic UI using EJS Templates</li>
  <li>⚡ Fast and Scalable Backend (Node.js & Express)</li>
</ul>

<h2>⚙️ Installation</h2>

<pre>
# Clone the repository
git clone https://github.com/your-username/cloudvault.git

# Navigate to project folder
cd cloudvault

# Install dependencies
npm install

# Run the server
npm start
</pre>

<hr/>

<h2>🔑 Environment Variables</h2>

<p>Create a <code>.env</code> file and add the following:</p>

<pre>
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
</pre>
